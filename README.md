# 🎵 Discord Music Bot

Музыкальный бот для Discord с поддержкой YouTube и SoundCloud.

## ✨ Возможности

- ✅ Воспроизведение музыки с YouTube и SoundCloud
- ✅ Поиск треков по названию
- ✅ Умная очередь треков
- ✅ Красивые Embed сообщения
- ✅ Регулировка громкости
- ✅ Перемешивание очереди
- ✅ Информация о текущем треке

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `!play <запрос/URL>` | Воспроизвести музыку |
| `!join` | Подключиться к каналу |
| `!stop` | Остановить и отключиться |
| `!skip` | Пропустить трек |
| `!pause` | Пауза |
| `!resume` | Продолжить |
| `!queue` | Показать очередь |
| `!clear` | Очистить очередь |
| `!shuffle` | Перемешать очередь |
| `!volume [0-100]` | Изменить громкость |
| `!np` | Текущий трек |
| `!help_music` | Показать справку |

## 🚀 Установка на Bothost

1. **Создайте репозиторий на GitHub** и загрузите все файлы

2. **На Bothost:**
   - Подключите GitHub репозиторий
   - Укажите команду запуска: `python bot.py`
   - Добавьте переменную окружения `DISCORD_TOKEN` с токеном бота

3. **FFmpeg** уже установлен на Bothost!
